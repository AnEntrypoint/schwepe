## Playwright MCP Investigation

### Findings
**MCP Configuration:** Found in `.mcp.json`
```json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["@playwright/mcp@latest", "--browser=chrome", "--viewport-size=1920x1080"]
    }
  }
}
```

**Status:** ❌ Not functional in Claude Code Web

**Reason:** 
- Claude Code Web/Mobile only supports **remote** MCP servers
- Local MCP servers (like the one configured) require spawning processes on the local machine
- The current config uses `npx` which requires local execution
- Environment: `CLAUDE_CODE_REMOTE=true` indicates web/cloud environment

**Remote MCP Solution:**
Would require:
1. Deploying Playwright MCP as a remote HTTP/SSE service
2. Configuring via claude.ai website (not `.mcp.json`)
3. Infrastructure for authentication and network access

**Current Best Practice:** Use Playwright CLI directly via Bash commands
